TI-99 BASIC Solutions
--------------------------------------------------------------------------------
Solutions in this directory are implemented in [TI BASIC], an ANSI-compliant
implementation of [BASIC] developed for Texas Instruments' [TI-99/4] home
computer, an early 16-bit system based on the TI-990 derived [TMS9900]
microprocessor.
