FOCAL Solutions
--------------------------------------------------------------------------------
Solutions in this directory are implemented in [FOCAL], an early BASIC-like
programming language for the famous [DEC PDP-8] minicomputer and other 
similarly low-cost DEC machines.

FOCAL is a primitive interpreted language based on the [JOHNNIAC Open Shop 
System], itself one of the first interactive programming systems, and oriented
towards high-precision mathematical work on memory-constrained systems. FOCAL
lacks many modern language features such as functions and all but the most
basic of loop and branching constructs, making it a fun challenge for more
serious number crunching.
